## PaPyPaint
PaPyPaint helps people to draw their pictures in some basic shapes like lines, rectangles, circles, etc. And color them.

### Core Features
* Draw shapes
* Choose color, thickness, ...
* Undo, redo
* Save to png, jpg
* Load old picture
